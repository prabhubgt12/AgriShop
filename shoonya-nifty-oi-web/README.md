# Shoonya NIFTY OI Web (Local)

## Setup

1. Ensure you have cloned Shoonya SDK:

- `D:\Shoonya\ShoonyaApi-js`

2. Edit `.env` in this folder and fill your Shoonya credentials.

## Run

### Option A (double-click)

- Run `start-web.cmd`

### Option B (terminal)

- `npm install`
- `npm start`

Open:

- http://localhost:3000

## Login (SMS OTP)

- Enter OTP in the UI and click **Login**
- Then click **Start**
